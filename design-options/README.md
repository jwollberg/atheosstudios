# Atheos Studios — Website Design Mockups

Three standalone concept mockups for **Atheos Studios** (atheosstudios.com), an independent video game studio. Each is a single, self-contained HTML file that opens by double-click — all visuals are built from CSS gradients, CSS shapes, and inline SVG (no external images, no build step, works fully offline). Every option is responsive (mobile-first, tuned at 375px and 1440px), keyboard-accessible with visible focus states and AA contrast, respects `prefers-reduced-motion`, and features the studio's first game **Grave Error** (undead survival horror, "Coming soon") with a clear button linking to https://graveerrorgame.com.

Open any file directly in a browser to preview. A fixed corner badge marks each as a preview mockup.

## Option A — Nebula (`option-a-nebula.html`)
Dark and cinematic. A deep-space backdrop with drifting stars, glowing gradient planets, and oversized gradient display type carry a single neon accent (cyan into violet). The mood is mythic and vast — a wordmark that feels like a title card, a Grave Error feature card with moonlit CSS/SVG key-art, and soft glow throughout. Best if the studio wants an atmospheric, AAA-trailer feel. Type: Sora + Space Grotesk.

## Option B — Monolith (`option-b-monolith.html`)
Bold editorial / brutalist. Near-black ink on warm off-white "paper," enormous uppercase Archivo Black headlines, a strict visible grid with hard rules and boxed spec tables, and one blood-red accent. A scrolling marquee strip and hard-shadow buttons give it a printed-poster confidence. Best if the studio wants to feel severe, opinionated, and instantly memorable. Type: Archivo Black + Archivo + IBM Plex Mono.

## Option C — Vector (`option-c-vector.html`)
Sleek and minimal-futuristic. Mostly black with thin hairline rules, a faint technical grid, a rotating orbit diagram, and HUD-style coordinate labels — restrained electric-lime accent, lots of negative space. The Grave Error key-art is a precise wireframe headstone with a rising light beam. Best if the studio wants a quiet, engineered, sci-fi-precise identity. Type: Chakra Petch + Inter.
